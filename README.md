# Práctica Conjunta - Proyecto Final Integrador

¡Bienvenidos a nuestro proyecto!

Este repositorio contiene la práctica final unificadora diseñada para consolidar y demostrar los conocimientos adquiridos durante el segundo año de Formación Profesional en la **Escuela de Arte Granada**. El proyecto integra las competencias de las siguientes asignaturas:

- **Desarrollo Web en Entorno Servidor**
- **Desarrollo Web en Entorno Cliente**
- **Diseño de Interfaces Web**

## Tecnologías Utilizadas

Este proyecto se ha desarrollado utilizando las siguientes tecnologías punteras:

- **Symfony**: Framework de PHP para el desarrollo robusto del Back-end.
- **React**: Biblioteca de JavaScript para la construcción de interfaces de usuario dinámicas en el Front-end.
- **Tailwind CSS**: Framework de CSS para un diseño rápido y personalizado.
